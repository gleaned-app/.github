  # gleaned

  A personal learning journal — end-to-end encrypted, self-hosted, open source.
  
  Write down what you learn each day. gleaned surfaces it again right before you'd forget it, using the FSRS-5 spaced
  repetition algorithm. No accounts, no cloud, no subscription. One Docker command and it's yours.
  
  → **[gleaned.dev](https://gleaned.dev)** · [Quick start](https://gleaned.dev/docs) ·
  [GitHub](https://github.com/gleaned-app/gleaned)

  | | |
  |---|---|
